Welcome

DataHabit is one of those rare weekend projets that become useful. It's a pretty straightforward flexible datalogger for Android. You can set up reminders, and export your data as a spreadsheet.

I'm putting it up on GitHub because it's big enough now that I need good version control (and it never hurts to have another backup). I'd welcome any collabortaors - fork it, baby! Make your own, or contribute back to the main branch. Whatever. I'm not greedy
